## вҡ”пёҸ ВҝQuГ© es The Data Machine?

> *"Millones de reseГұas. Cero tiempo para leerlas. Una mГЎquina para entenderlas todas."*

**The Data Machine** es una aplicaciГіn web de anГЎlisis de reseГұas de videojuegos. A partir de un dataset real de Steam con millones de opiniones, la app clasifica el sentimiento de la comunidad, detecta los temas mГЎs recurrentes y lo presenta todo en un dashboard interactivo вҖ” sin que el usuario tenga que leer ni una sola reseГұa.

El usuario inicia sesiГіn, elige su juego del catГЎlogo y obtiene en segundos:

- рҹ“Ҡ **ВҝLa comunidad lo ama o lo odia?** вҖ” grГЎfica de sentimiento positivo / negativo / neutral
- вҳҒпёҸ **ВҝDe quГ© habla la gente?** вҖ” nube de palabras con los tГ©rminos mГЎs frecuentes
- рҹҸ·пёҸ **ВҝQuГ© temas se repiten?** вҖ” bugs, historia, precio, multijugador... lo que mГЎs menciona la gente
- в¬ҮпёҸ **ВҝQuieres los datos?** вҖ” descarga el anГЎlisis completo en CSV o la grГЎfica en PNG
